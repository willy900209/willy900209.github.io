About Me

我是蔡詔崴，畢業於國立臺北大學統計研究所，具備統計分析、資料處理、機器學習建模與影像資料分析經驗。 研究所期間，我的研究主題聚焦於日冕洞演化對太陽風速變化之影響，主要結合太陽盤面影像處理、 日冕洞特徵擷取、時間序列資料整理與機器學習模型，建立太陽風速預測與分析流程。

在研究過程中，我使用 Python 進行大量資料處理與模型建構，包含影像前處理、遮罩建立、特徵擷取、 時間滯後特徵設計、資料合併、模型訓練與預測結果評估。曾使用 pandas、NumPy、matplotlib、OpenCV、 scikit-learn 與 XGBoost 等工具，處理太陽影像資料與太陽風時間序列資料，並進行特徵工程、 模型比較與結果視覺化。

除了 Python 外，我也具備 R 語言資料分析能力，能進行資料清理、描述性統計、假設檢定、 迴歸分析與圖表呈現；SQL 方面則具備基礎查詢能力，可進行資料篩選、分組彙整與資料表合併。 整體而言，我具備從資料前處理、特徵建構、統計分析、模型訓練到結果解釋的完整資料分析經驗。

未來希望投入資料分析、統計分析、機器學習建模或研究資料處理相關職務， 將統計背景與程式能力應用於實務資料分析問題，協助團隊從複雜資料中建立可解釋且具應用價值的分析結果。
